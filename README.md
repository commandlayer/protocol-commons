The CommandLayer Protocol — Commons defines the **canonical verb language**
for A2A (agent-to-agent) communication.  
It is the universal foundation for agents, LLMs, SDKs, workflows, and autonomous systems.

The Commons is aligned with **x402** and compatible with **ERC-8004** schema discovery.

# Canonical Verb Set (v1.0.0)

| Verb      | Aliases (max 2)          |
|-----------|---------------------------|
| analyze   | examine, inspect          |
| classify  | categorize, label         |
| clean     | sanitize, normalize       |
| convert   | transform, cast           |
| describe  | characterize, depict      |
| explain   | clarify, justify          |
| format    | reformat, style           |
| parse     | extract, decode           |
| summarize | condense, outline         |
| fetch     | retrieve, get             |

Aliases are optional user-friendly synonyms.  
Canonical verbs always dominate protocol semantics.
